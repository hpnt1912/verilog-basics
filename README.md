# 🚀 Verilog Digital RTL Design Journey

## 👤 About Me
First-year student at UIT (VNU-HCM), learning digital logic design and Verilog HDL with the goal of pursuing IC Design / VLSI.

---

## 🎯 Goal
Build strong fundamentals in digital design:
- Combinational logic
- Sequential logic
- Finite State Machines (FSM)

and learn how to design, simulate, debug, and verify digital circuits using Verilog.

- Git & GitHub (version control)

## 🧪 Learning Workflow

Write RTL (VS Code)
→ Simulation (Quartus / ModelSim)
→ Check waveform
→ Debug & fix
→ Re-simulate
→ Commit to GitHub

✔ Check Quartus warnings before commit
